# Calculadora do Biel (feita com Claude)

Calculadora web estática com interface responsiva, histórico de operações e suporte a tema claro/escuro.

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Porcentagem e inversão de sinal (+/−)
- Encadeamento de operações sem precisar pressionar `=`
- Histórico de operações com horário — clique em um item para reutilizar o resultado
- Tema claro/escuro com toggle no topo e persistência via `localStorage`
- Suporte a teclado (dígitos, operadores, `Enter`, `Backspace`, `Escape`)
- Layout responsivo para desktop e mobile
